Calculo Equação de Segundo Grau

Insira a equação abaixo, utilize:

  • [+] para adição
  • [-] para subtração
  • [*] para multiplicação
  • [/] para divisão
  • [^] para exponenciação
  • [x] para incógnita
  • Insira a formula igualada a 0, seguindo a ordem x^2 + x^1 + x^0
  • Exemplo: "15x^2 + 25x + 5 = 0"
